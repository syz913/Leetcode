> 题目描述：幂集。编写一种方法，返回某集合的所有子集。集合中**不包含重复的元素**。
>
> 说明：解集不能包含重复的子集。

eg:

```java
示例:

 输入： nums = [1,2,3]
 输出：
[
  [3],
  [1],
  [2],
  [1,2,3],
  [1,3],
  [2,3],
  [1,2],
  []
]
```

> 思路描述：
>

```C++
class Solution {
public:
    vector<vector<int>> v;
    vector<vector<int>> subsets(vector<int>& nums) {
        v.push_back({});
        for(int i = 0 ; i < nums.size() ; i++){
            vector<vector<int>> b = v;
            for(int j = 0 ; j < b.size() ; j++){
                vector<int> a = v[j];
                a.push_back(nums[i]);
                v.push_back(a);
            }
        }
        return v;
    }
};

```
