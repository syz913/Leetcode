> 题目描述：一只青蛙一次可以跳上1级台阶，也可以跳上2级台阶。求该青蛙跳上一个 `n` 级的台阶总共有多少种跳法。
>
> 答案需要取模 1e9+7（1000000007），如计算初始结果为：1000000008，请返回 1。

eg:

```java
示例 1：

输入：n = 2
输出：2
示例 2：

输入：n = 7
输出：21
```

> 思路描述：略
>

```C++
class Solution {
public:
    int numWays(int n) {
        if(n <= 1) return 1;
        if(n == 2) return 2;
        int f1 = 1, f2 = 2;
        for(int i = 3; i <= n; i ++){
            int temp = f2 % 1000000007;
            f2 = f1 % 1000000007 + f2 % 1000000007;
            f1 = temp;
        } 
        return f2 % 1000000007;
    }
};
```
